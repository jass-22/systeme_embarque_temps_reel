Projet : Système embarqué en temps réel

Ce projet implémente un système embarqué en temps réel permettant de modifier les paramètres PWM (Pulse Width Modulation) via une interface graphique développée avec Node.js.
Contenu du projet

    -pwm.html : Interface utilisateur pour contrôler les paramètres PWM.
    -pwm.sh : Script pour gérer la configuration du système.
    -pwm.jjjs : Fichier JavaScript pour les opérations logiques et les interactions.
    -Vidéo démonstrative : Une vidéo illustrant le fonctionnement du projet.

Fonctionnalités principales: 
    -Modification des paramètres PWM en temps réel.
    -Utilisation d'une interface graphique intuitive et accessible.
    -Intégration avec des technologies modernes pour un contrôle efficace.
